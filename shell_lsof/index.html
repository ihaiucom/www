【笔记】MAC查看端口占用情况

2016-08-16 | 发布 大海明月  zengfeng75@qq.com  | 分类 shell  | 标签 mac  shell 

命令 lsof -i tcp:port （port替换成端口号，比如6379）可以查看该端口被什么程序占用，并显示PID，方便KILL

lsof -i :80
sudo lsof -n -P| grep :8087
nc -v -w 2 127.0.0.1 8087
lsof -i tcp:8087
