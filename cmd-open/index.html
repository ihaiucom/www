扩展Windows命令open

2018-01-06 | 发布 大海明月  zengfeng75@qq.com  | 分类 cmd  | 标签 cmd 


扩展windows系统 open命令。类似Mac系统的open命令



工具下载

本地下载

Github

安装

双击 install.bat 会执行下面的操作

1. 复制 System32 -> C:\Windows\System32 2. 复制 Git\usr\bin -> D:\Program Files\Git\usr\bin


使用

在window系统不管是cmd 还是 git里的 shell 都可以使用

# 测试打开文件,相对路径
open link.txt

# 测试打开文件,绝对路径
open /c/zengfeng/githbug/ihaiu.blog/index.html


# 测试选中文件,绝对路径
open -R /c/zengfeng/githbug/ihaiu.blog/index.html


# 测试选中文件,绝对路径 windows路径风格
open -R c:/zengfeng/githbug/ihaiu.blog/index.html

# 测试选中文件夹,绝对路径
open -R /c/zengfeng/githbug/ihaiu.blog
